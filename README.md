# Waryen Plugins

My personal [Claude Code](https://claude.ai/code) plugin marketplace. Claude Code plugins extend your AI assistant with new skills and workflows — this marketplace packages them so you can install them in any project with a single command.

## Getting started

Add this marketplace to Claude Code:

```
/plugin marketplace add Waryen/waryen-plugins
```

Then install any plugin:

```
/plugin install <plugin-name>@waryen-plugins
```
